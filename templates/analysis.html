Trading Agent

Hi, this is a trading simulation environment where I play with different reinforcement learning algorithms and apply them to solve a simple trading problem. I'm getting data from iexcloud where I'm using the free tier to expose this data using their sandbox environment. Therefore DO NOT make any real trading decisions using this simulator. Currently using a temporal difference based Q-Learning algorithm.
